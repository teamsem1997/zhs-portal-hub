# zhs-portal-hub

本项目用于归档和发布多个独立的 HTML 页面。这些页面不针对任何特定域名或网站，仅作为通用网页资料的存储与分发仓库。

## 项目简介

zhs-portal-hub 是一个静态 HTML 页面集合，汇集了多种独立功能或内容的网页文件。每个页面均为自包含的 HTML 文件，可直接在浏览器中打开使用，无需额外服务端支持。

## 目录说明

```
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 公共资源文件（如 CSS、JS、图片等）
├── index.html      # 仓库入口页面（可选）
└── README.md       # 本文件
```

- `pages/`：主要页面归档目录，每个子目录或文件代表一个独立的 HTML 页面。
- `assets/`：存放页面可能引用的公共资源，避免重复文件。
- `index.html`：可选入口页面，用于统一展示或导航。

## 页面归档说明

本仓库中的页面内容多样，涵盖工具、演示、信息展示等类型。所有页面均保持独立，不依赖外部服务或特定域名。您可以直接使用 `pages/` 下的文件，或通过 GitHub Pages 等方式部署访问。

页面命名遵循清晰、直观的原则，便于识别和查找。每个页面通常附带简要说明（可内嵌于 HTML 注释或单独文档）。

## 维护说明

- **新增页面**：请将新 HTML 文件放入 `pages/` 目录，并确保引用资源路径正确。
- **更新页面**：直接修改对应文件，注意保持页面兼容性和独立性。
- **删除页面**：移除相关文件及引用链接，保持仓库整洁。
- **资源管理**：公共资源请统一存放于 `assets/`，避免重复。

欢迎提交 Issue 或 Pull Request 来完善页面内容或提出改进建议。本仓库不承诺特定更新频率，但会尽力维护已有页面的可用性。
